# API List for J-Hub Front End

## Active API base
- `http://192.168.1.50:5000` (fixed LAN backend URL)

## API calls currently used in code
1. `GET /jobs`
   - Purpose: Fetch all jobs displayed in the main jobs feed.
   - Called from: `App.js` in `fetchJobs`.
   - Expected response formats supported by app:
     - Array of jobs: `[{...}, {...}]`
     - Object containing jobs array: `{ "jobs": [{...}, {...}] }`

## Job fields consumed by frontend (with fallback defaults)
- `id` or `_id`
- `title`
- `company` or `companyName`
- `salary`
- `deadline` or `applicationDeadline`
- `postedDaysAgo` or `postedAt` (converted to day count)
- `location`
- `employmentType`
- `level`
- `description`
- `responsibilities[]`
- `requirements[]`

## API calls not yet implemented (still local UI behavior)
- Bookmark create/delete/fetch endpoints
- Apply-to-job endpoint
- Auth/profile endpoints
